Useful Resources

This page contains useful resources and information about how to learn HTML, CSS and design. The sites have been useful when creating this website and some of them would be very useful for future projects.

CSS Tricks Website

This website Contains very useful information about css in general, specifically it was good when implementing the flexbox. View the flexbox guide here

FontAwesome Website

This website is the home of FontAwesome, an icon set that is free to use and allows people to make more engaging sites through the use of icons. View the resource here

GitHub Website

GitHub is a place where you can create projects that are available for everyone to see. You can buy private space as well in case you're working on something you don't want others to see. View the resource here

Material Design Documentation Website

This website hosts the entire Material Design Specification and is updated and maintained by Google. View the resource here

Materializecss Website

This is the home of the materializecss framwork for web developers wanting to adhere to the Material Design specification. It makes the simple things easier to do although I have not used it in this site it would be a good resource for other projects. View the resource here

Uplabs Website

This website contains many useful examples that you can draw from to create your own site. View the resource here

W3Schools Website

This website contains useful tutorials for learning about HTML and CSS. View the resource here
