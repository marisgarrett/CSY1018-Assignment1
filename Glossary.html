Here is a collection of useful web development terms and definitions.
Term Definition
Accessibility This is the word that is used to describe making a website available and useable to a person with disabilities.
AJAX Asynchronous JavaScript and XML is a client side function that is used to allow parts of a web page to be dynamically populated.
Back End This is the part of a website a user of the site never sees. It includes things like the structure of the site and would include a CMS if one was used.
Bandwidth This means the speed at which data can be transferred accross the internet. A site that uses too much bandwidth would be difficult for people to use on mobile devices or slow internet connections.
Below the Fold This is an old term that used to refer to newspapers when they were folded in half on the stand, the content on the second half of the page was refered to as being below the fold. It now uses the same principal but applies to content that is hidden below the first page ie. content that must be scrolled to.
Breadcrumbs These are essentially a path to your location in the site. It allows the user to see where they are easily as well as allowing them to navigate upwards in the site structure more easily.
Browser This is a shortened term for the Web Browser, it is the program that you use to access the websites on the internet.
Cache A browser keeps copies of files locally so that it doesn't have to download them over and over again. This improves the speed that users can view your site.
Cascading Style Sheets CSS stylesheets are in control of how your site looks. They drive all of the layout of the HTML you use. CSS should be kept in separate files to the HTML to allow you to completely change the look and feel of your site from central location without having to search through all of your files.
Content Delivery Network A CDN is tha system designed to deliver content to users quickly from locations nearby. Another benefit is that if you're using shared libraries your users might already have the files in their browsers and then not need to download them again for your site. This would improve the speed your site loads at.
Client Side When something is client side it is run on the users computer not on your server. JavaScript is an example of this, it's run the in the browser.
Content Management System This is a system that is often used for websites to enable less technical users to maintain and add content. Wordpress is a very popular and widely used CMS.
DNS A Domain Name Service server is the actual server that converts an IP address into a domain name and also does the reverse (A domain name into an IP address).
DOCTYPE This tells the browser how to interpret the pages in your website, and which version of HTML to use.
Domain A domain is an identifier for a website, it allows your customers to naviagte to your site using a friendly name instead of an IP address.
Favicon This is the icon that appears as your sites favourite icon in most browsers.
Flexbox This is a CSS technology that is used to provide more flexible layout rendering in the browser. It has been specifically designed for this purpose.
Hex Code A hex code is the identifier for a colour. For example #000000 is black. It is used by browsers to create a colour on the users screen.
HTML Hyper Text Markup Language (HTML) This is what web pages are written in.
HTTP Hyper Text Transfer Protocol (HTTP) is the protocol used to transmit website accross the internet.
HTTPS Hyper Text Transfer Protocol over SSL. This is the secure version of the HTTP protocol. It encrypts all data before it is transmitted over the internet allowing for secure communications to take place.
Hyperlink This is the way a web page links to another page.
Inline Style This is the term used to describe a style that is applied directly to an element instead of through an external CSS file.
IP An IP address is the unique identifier for a resource on the web.
ISP An Internet Service Provider is the service that gives you a connection to the internet.
JavaScript This is a client side scripting language that allows you to manipulate html content in the browser. It also provides other rich functionality such as timers.
JPG/JPEG A JPG/JPEG is a common format for an image to be used in a web page to be in.
JSON This is a content transfer syntax designed to be as lightweight as possible to allow fast cross platform data transfer. It is a faster equivalent to XML.
Landing Page This is the page that a visitor first arrives on your website at.
Media Queries These are queries that allow you to change the CSS styles that are applied to a page when certain rules are met. For example a screen that is wider than 200px could have additional style rules applied.
Meta Data Data added to a website that defines additonal information about it. You can't actually see the content on the page but it is used by other systems like search engines.
Mobile First A way of developing web applications to ensure they work on a mobile device and a larger device with minimal effort.
Nesting This is a term used to describe putting a HTML element inside another one. For example a p tag inside a div tag.
Open Source Open Source is a term used to define the fact that the code used to build the site/application is available for people to see.
Optimise This is the process of making files, images and other content that is going to be served on your website as small as possible to make them as fast as possible.
Pixel A pixel is a measurement on the screen and is used heavily in CSS to define the size of elements.
Property A property is used within CSS and is essentially a definition of how something should behave.
Responsive Design This is a commonly used practice that allows a web page to adapt to the size of the screen it's being viewed on.
REST Representational State Transfer is a standard that defines a way for applications to communicate with each other over the internet.
Really Simple Syndication RSS is a standard used to provide data feeds to users. They subscribe to it and receive updated simple versions of content.
Selector This refers to the item that a CSS style would be applied to.
SEO Search Engine Optimisation is a process that you can use to make your website appear as high in the search engine results as possible.
Server Side These are scripts that run on the server instead of the browser. When a server side script executes a new page would have to be sent to the browser.
UI User interface is the part of your applications that the user interacts with. It is often combined with the term UX.
Upload The process of adding a website to a web server to make it available on the internet is often called uploading. More generally it can mean adding content to a server/service.
Url A Uniform Resource Locator is a link to where something can be found on the internet.
Usability This is a term used to describe how usable a site is for a user. Is it easy and intuitive for them to learn how to use for example.
UX User Experience is the process of evaluating how a user uses your site to ensure that they have the best experience possible.
WWW The World Wide Web is another name for the internet.
XML Extensible Markup Language is a language that is commonly used to let different applications talk to each other. This works by all devices following the same syntax.
